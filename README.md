# Keyfinder: App Info & Shortcut Management Utility

Keyfinder is a professional-grade utility designed for developers and power users to capture, manage, and analyze Windows application information. It streamlines the process of tracking system metadata and accessibility features.

## ‚ú® Key Features
- **App Metadata Capture:** Instantly retrieve detailed information about active Windows applications.
- **Database Integration:** Built-in SQLite support for logging and tracking application data over time.
- **Stable Responsive UI:** Advanced Dark/Light mode support with synchronized system theme detection to ensure readability.
- **Accessibility Tracking:** Tools specifically designed to monitor and log accessibility compatibility across different software.
- **Export Capabilities:** Export captured data for use in documentation or technical audits.
